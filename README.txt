#--- SPÚŠŤANIE WEBU ---#

Web spustíš napr. príkazom

php -S localhost:4200

a následne do prehliadača zadáš "localhost:4200".
Druhá možnosť je iba otvoriť index.html a stránka by sa mala zobraziť.
#----------------------#


#--- POST NA GIT ---#

Keď budeš so svojím webom spokojný, zaves ho na github príkazmi:

git add .
git commit -m "final version"
git push

Aby ti to fungovalo, musíš byť v správnom adresári. Príkazom pwd zistíš, kde sa nachádzaš.
